# 📌 Overview

VectorCommerce is a modern full-stack MERN E-Commerce platform that provides a complete online shopping experience.

The project consists of three independent applications:

- 🛍️ Customer Storefront
- ⚙️ Administrator Dashboard
- 🖥️ Express.js REST API Server

Customers can browse products, search, manage cart, place orders, update profile, and manage their account.

Administrators can manage products, categories, users, orders, sales analytics, inventory, and AI-powered search embeddings.

# ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Go to project

```bash
cd VectorCommerce
```

Install dependencies

```bash
npm install
```

---

# 🚀 Running the Project

## 1️⃣ Start Backend Server

Open Terminal 1

```bash
npm run dev-server
```

Backend Health API

```
http://localhost:5000/api/health
```

Expected Response

```json
{
  "status": "healthy"
}
```
